# SpeechBuddy

SpeechBuddy is an Android mobile application designed to help users improve their English pronunciation through interactive speech learning exercises.

## Features
- Category-based learning (Animals, Colors, Numbers, Food)
- Pronunciation practice
- User progress tracking
- Simple and friendly UI

##  Technologies Used
- Java
- Android Studio
- XML
- Git & GitHub

##  Purpose
This project was developed as part of an academic project to support speech and pronunciation learning.
